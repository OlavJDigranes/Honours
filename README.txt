APPLICATION USAGE FOR TEST SUBJECTS:
    1: Run the executable (Honours.exe) once. Ensure system audio is on and audible, that one keyboard is connected and that one mouse is connected. 
    2: Complete one test level, input values at the end of level questionnaire. 
    3: Repeat step 2 until a black screen with a big "EXIT" button appears. Click said button to exit program. 
    4: Locate the Data.csv file in the Honours_Data directory. Return the file to the test provider. 

APPLICATION USAGE FOR TEST PROVIDERS:
    1: Provide test subjects with test application. 
    2: Record returned data. 
    3: Analyse retruned data.
